# 공동 스터디 페이지

1. 먼저 깃 저장소를 로컬에 클론합니다!
	1. https://github.com/co-study/co-study.github.io.git
2. Obsidian을 설치해서 클론한 저장소를 보관함 열기로 연결합니다. 
  <img width="1200" height="975" alt="image" src="https://github.com/user-attachments/assets/1c6db42c-0fdc-44da-8bc0-64ea9c5258b3" />
3. 옵시디언이 켜지면 좌하단에 톱니바퀴를 눌러 "커뮤니티 플러그인" 탭으로 이동합니다.
  <img width="1543" height="1207" alt="image" src="https://github.com/user-attachments/assets/16f8a6f4-f8f5-4888-9909-adc590bcfe86" />
4. 그 다음 활성화를 합니다.
  <img width="1543" height="1207" alt="image" src="https://github.com/user-attachments/assets/888ac9ac-91b7-4e47-9c6a-cede16eebd89" />
5. "커뮤니티 플러그인" 옆 `탐색`버튼을 클릭하고 Git을 검색하여 설치합니다.
6. 설치된 후 `활성화` 버튼으로 Git 플러그인을 활성화합니다.
7. 이제 docs 디렉토리 아래에 자신이 진행한 스터디를 "연/월/일/파일명.md"로 작성해서 아래 이미지와 같이 "commit and sync"를 클릭하면 작성한 내용이 배포됩니다.
  <img width="1543" height="1207" alt="image" src="https://github.com/user-attachments/assets/d6df8602-4815-4ded-af9c-3163d706ab00" />
